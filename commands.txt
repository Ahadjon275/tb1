/help - Справка
/config - Меню настроек
/lang - Меняет язык локализации, автоопределение по умолчанию
/style - Стиль ответов
/reset - Стереть память
/undo - Стереть только последний запрос
/tts - Преобразует текст в речь и отправляет аудиосообщение, с указанием языка и скорости (/tts [ru|en|uk|...] [+|-50%] hello all)
/trans - Переводит текст на другой язык (/trans en привет как дела)
/image - рисует картинки по текстовому описанию
/bing - рисует с только DALL-E 3 от bing.com
/image2 - рисует картинки по текстовому описанию без цензуры
/image10 - 10 раз подряд рисует картинки по текстовому описанию
/stop - остановить генерацию изображений
/google - бот будет гуглить перед ответом
/name - меняет кодовое слово для обращения к боту (только русские и английские буквы и цифры после букв, не больше 10 всего)
/sum - делает выжимку из текста по ссылке (сумморизацию текста) /sum URL
/sum2 - то же что и /sum но не берет ответы из кеша, повторяет запрос заново
/ask - отвечает на запрос по сохраненному тексту, текст сохраняется при запросах в гугл и пересказе файла/ссылки
/ocr - настройки языка для распознавания текста с картинок
/temperature - уровень бреда от 0 до 2
/original_mode - вкл-выкл подсказки для бота о том кто он, где и когда находится, с кем говорит и какую роль играет
/mem - показывает содержимое своей памяти, в отладочных целях
/purge - удалить мои логи
/openrouter - выбрать модель от openrouter.ai
/model - изменить модель openrouter.ai
/maxtokens - изменить количество токенов для openrouter.ai
/maxhistlines - изменить количество запросов (размер памяти) в памяти для openrouter.ai
/maxhistchars - изменить количество символов (размер памяти) в памяти для openrouter.ai
/free - как и почему бесплатно
/id - показывает id чата
/remove_keyboard - удалить клавиатуру
/keys - вставить свой API ключ в бота
/report - сообщить о проблеме
